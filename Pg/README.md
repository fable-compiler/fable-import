Fable bindings for [node-postgres](https://github.com/brianc/node-postgres).
=======
[![nuget](https://badge.fury.io/nu/Fable.Pg.svg)](https://badge.fury.io/nu/Fable.Pg)



### Using
1. `paket add Fable.Pg`
2. `yarn add pg`

### Building the package
1. `dotnet restore`
2. `dotnet build`
3. `dotnet pack`
